## Release Description

```markdown
# MosaicHub Portal v1.0.0

First public release of MosaicHub Portal, a modern enterprise portal and AI-powered CMS built with Next.js 14, React 18, TypeScript, Prisma and MySQL.

## Highlights

- Modular homepage builder with carousel, quick navigation, category feeds, image cards and custom HTML modules
- Complete admin dashboard for posts, comments, users, navigation menus, site settings and analytics
- Role-based access control with SUPER_ADMIN, ADMIN and USER roles
- AI assistant, AI layout designer, AI writing helper and AI-generated HTML page support
- Multi-provider AI configuration for OpenAI, DeepSeek, Qwen, Kimi, GLM, Gemini, Claude and OpenAI-compatible APIs
- Dynamic theme presets, dark mode and per-user theme preferences
- Comment system with nested replies, emoji support, reactions and sensitive word filtering
- Secure architecture with JWT authentication, encrypted AI keys, upload validation and sandboxed HTML rendering
- Bilingual documentation: English README and Simplified Chinese README

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- TailwindCSS
- Prisma ORM
- MySQL
- JWT Authentication
- AES-256 encryption

## Quick Start

```bash
npm install
npx prisma db push
npx prisma db seed
npm run dev
```

Open:

```text
http://localhost:9055
```
